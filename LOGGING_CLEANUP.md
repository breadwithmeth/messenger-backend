## Преимущества

1. **Меньше шума в логах** - только важные входящие сообщения от клиентов
2. **Легче мониторинг** - проще отследить новые обращения
3. **Производительность** - меньше операций I/O для логирования
4. **Чистый продакшн** - логи не захламляются техническими деталями

## Возврат логов

Если потребуется вернуть логи для отладки, просто раскомментируйте нужные строки в `src/config/baileys.ts` (они помечены `//`).

## Полное отключение логов

Все логи приложения можно отключить через env-флаг:

```env
APP_LOGS_ENABLED=false
```

Этот режим переводит `pino` в `silent` и глушит прямые `console.*` вызовы, включая request/webhook/QR-логи. Для совместимости также поддерживаются `APP_DISABLE_LOGS=true` и `DISABLE_LOGS=true`.
